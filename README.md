# 🏡 StayPoint

StayPoint is a full-stack web application inspired by Airbnb where users can explore, create, edit, and review property listings.

The project is built using the MVC architecture and focuses on authentication, authorization, CRUD operations, image uploads, and secure backend development.

---

## 🚀 Features

- User Authentication (Signup/Login/Logout)
- Authorization
- Create Listing
- Edit Listing
- Delete Listing
- View Property Details
- Add Reviews
- Delete Reviews
- Image Upload
- Flash Messages
- Form Validation
- Responsive UI
- Error Handling

## ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Move into the project

```bash
cd StayPoint
```

Install dependencies

```bash
npm install
```

Create a `.env` file and add:

```env
ATLASDB_URL=your_mongodb_connection
SECRET=your_secret
CLOUD_NAME=your_cloud_name
CLOUD_API_KEY=your_api_key
CLOUD_API_SECRET=your_api_secret
```

Run the project

```bash
npm start
```

Visit

```
http://localhost:3000
```
